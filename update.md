# Update

This tab has a text input to enter the **Download ID** provided by **[J2Store](http://j2store.org)**, which is a requirement for updating the j2store component's PRO version. Without this ID, [j2store PRO](http://j2store.org/my-downloads) cannot be downloaded. This ID will be available in the Invoice as well as in **My Account** of [j2store.org](http://j2store.org)
